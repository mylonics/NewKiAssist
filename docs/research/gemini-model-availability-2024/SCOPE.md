# Research Scope: Gemini Model Availability (Nov-Dec 2024)

## Objective
Determine availability, model IDs, API endpoints, and stability status for Google Gemini 2.5 and 3.0 model series.

## Task Requirements
1. Verify existence of Gemini 2.5 models (2.5-flash, 2.5-pro)
2. Verify existence of Gemini 3.0 models (3.0-flash, 3.0-pro)
3. Identify correct API endpoint version (v1 vs v1beta)
4. Determine release status (experimental/preview/stable)
5. Provide exact model ID strings for API calls

## Constraints
- Focus on models released November-December 2024
- Exclude Gemini 1.5 series from consideration
- Provide only factual, referenced information

Research conducted: November 22, 2025
Target model releases: November-December 2024
